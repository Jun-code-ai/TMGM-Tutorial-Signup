# HOME豪门 × TMGM Copy Trading 操作流程

TMGM 跟单交易（Copier 端）完整操作手册 — 含实景截图对照。

## 内容

- **14 张幻灯片** — 从开户到获利提取，7 阶段全覆盖
- **实景截图** — TMGM 用户面板、Signal Trading 登录页、入金界面、手机 App 等
- **HOME豪门 定制 SOP** — 8 步简化流程
- **三级风控体系** — Level 1/2/3 按资金量级匹配风险

## 使用方式

直接在浏览器打开 `index.html`：

```

```

或部署到 Vercel / Netlify 获得在线链接。
